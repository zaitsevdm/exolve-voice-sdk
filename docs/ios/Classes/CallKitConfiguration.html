  useGenericHandleType

Set this to YES make CallKit UI display only name found by your handler.
CallKit uses contact info from address book and ignores provided caller name. We can use different handle type to suppress such behavior. Note that you will need access address book by yourself if you still want to display device contacts, and that Recents will still show caller info from Contacts regardless of this option.
